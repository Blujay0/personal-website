What I do

UI / UX

Expertise in crafting visually pleasing and user-friendly digital experiences using CSS frameworks such as Bulma, Material UI, and Semantic UI. Focus on comprehensive user interface (UI) designs that transform ideas into captivating visuals, including layouts, colors, typography, icons, etc.

Start-to-finish user experience (UX) design focusing on creating intuitive and efficient user journeys through research, wireframing, and prototyping with tools like Figma. Prioritization of enjoyable and seamless interaction for the end user.

Responsive Design

Web development that focuses on effortless adaptability to a diverse range of devices and screen sizes. Accessible and visually appealing digital content, whether viewed on a desktop, tablet, or smartphone.

Through techniques like media queries, Flexbox, and CSS Grid, the layouts are able to adjust gracefully, guaranteeing a user-friendly experience across the entire spectrum of devices.

Performance Optimization

Dedicated to delivering fast and efficient digital solutions. Specialty in fine-tuning websites and applications to achieve optimal speed and resource efficiency.

My approach involves code analysis, image optimization, SEO and implementation of techniques like lazy loading and caching. By prioritizing performance, users experience smooth and responsive interactions, leading to improved user satisfaction, enhanced search engine rankings, and reduced bounce rates.
