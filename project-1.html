Weatherly

A Modern Weather App

The Weatherly app is a user-friendly, serverless web application designed to provide up-to-the-minute weather information for any city, state, zip code, and country. With a sleek and intuitive interface, users can effortlessly search for their desired location, and Weatherly delivers comprehensive weather forecasts, including local time, date, current conditions, and temperature in degrees Celsius. This app ensures that users are always in the know about the weather, making it an indispensable tool for planning daily activities or staying updated on weather conditions worldwide. With Weatherly, staying informed and prepared for any weather is just a click away.

Technologies & Approaches

This project delved into the core technologies of web development, encompassing HTML, CSS, and JavaScript. Beyond the basics, it seamlessly integrates the Weather API through fetch calls, offering real-time weather data. It also boasts robust error handling mechanisms to ensure a seamless user experience. Furthermore, the project excels in JSON data manipulation, allowing for the extraction and presentation of weather information. In essence, it's a comprehensive exploration of web development fundamentals with a practical, real-world application.
